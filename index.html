Song Finder

This project was a website that allowed users to find certain songs in select video games by using filters. This project involved using Flask to interact with HTML to create a website that used a database. The main aim of this project was to create a tool that allowed users to make queries using preset filters.

The reason I started this project was due to a need to find certain songs in a video game where the name of the song was not known. The user utilises various true/false filters to reduce the search size, then the embeds within the website makes it easier to discover the song required by the user.

Technology used
  • Python
  • Flask
  • HTML
  • CSS
